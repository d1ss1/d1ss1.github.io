# 🎮 Угадай число

Простая браузерная игра на JavaScript.

## Как играть

1. Открой сайт
2. Введи число от 1 до 100
3. Нажми "Проверить"
4. Получи подсказку — больше или меньше

## Технологии

- HTML
- CSS
- JavaScript

## Играть
